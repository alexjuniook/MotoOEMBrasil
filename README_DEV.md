Desenvolvimento local (inicial)

1. Copiar .env.example para .env e ajustar
2. docker-compose up --build
3. Entrar em apps/api e rodar npm install
4. Rodar schema: psql DATABASE_URL -f packages/database/schema.sql

Notas:
- Este repositório é um esqueleto inicial; implementar endpoints, telas e automações é o próximo passo.
